# UBNT directions cheat sheet

```
    IN - Packets going from the vlan to WAN/LAN. From the IN(side) to out.
        Example: Allowing or disallowing connections made from vlan to other vlan(s) or the internet
        Example: Traffic going into the router and onwards
        Example: IN traffic entering the router from an interface (and later exiting via another interface)


    LOCAL - Packets going from the vlan to the router. What services to talk to on the router
        Example: Allow devices on the vlan to talk to DNS and DHCP on the router
        Example: LOCAL traffic entering the router and destined to router itself
```
